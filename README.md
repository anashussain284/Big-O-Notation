
# Big-O-Notation

Big O Notation course practical problems and explanation



## Order of growth

Order of growth of the Big-oh notations

- ```O(1)``` is best case
- ```O(n!)``` is worst case

| Big-Oh expression | Name |
| ------ | ------ |
| O(1) | Constant |
| O(log n) | Logarithmic |
| O(n) | Linear |
| O(n log n) | Linearithmic |
| O(n^2) | Quadratic |
| O(n^3) | Cubic |
| O(2^n) | Exponential |
| O(n!) | Factorial |
